to compile the code:

cd to 'src'

run 'make'

and then run './main' <dimacs filename'

output for pagerank is in 'rageRank_results.txt'
output for node label array is in 'node_labels.txt'
output for the histogram is in 'out_degree_histogram.txt'
output for the CSR to dimacs is in 'output.dimacs'

in the 'pdfs' folder, it will have the write up and the pdf of the histograms
